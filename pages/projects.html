My Projects

All projects are lorem ipsum variants and not real projects.

Todo App

The Todo App is a simple web application that allows users to create, edit, and delete tasks. The app is built using React and Node.js, and uses a Firebase database to store task data. Users can sign in to the app using their Google account and access their tasks from any device.

Weather App

The Weather App is a web application that allows users to check the current weather and forecast for any location. The app is built using React and the OpenWeatherMap API, and displays weather data in a user-friendly format. Users can search for a location by name or use their current location to get the weather.

Blog Website

The Blog Website is a personal blog that allows users to create, edit, and delete blog posts. The website is built using React and Node.js, and uses a Postgresql database to store blog post data. Users can sign in to the website using their email address and password and share their thoughts with the world.

Chat Application

The Chat Application is a real-time messaging app that allows users to chat with each other in private or group chats. The app is built using React, Node.js, and Socket.io, and uses a Firebase database to store chat data. Users can sign in to the app using their email address and password and start chatting with friends and family.
